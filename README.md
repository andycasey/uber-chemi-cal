`uber-chemi-cal`
----------------
Uber-calibration for chemical abundances (kind of).

Install and make paper
----------------------
From a terminal:

    git clone git@github.com:andycasey/uber-chemi-cal.git uber-chemi-cal
    cd uber-chemi-cal/paper
    make
